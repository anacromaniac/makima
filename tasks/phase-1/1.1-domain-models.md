# Task 1.1: Domain Models and Enums

## Objective
Define all domain structs, enums, and error types in the `domain` crate. This is the vocabulary of the entire project ‚Äî all other crates depend on these types.

## Steps
1. Add `serde`, `chrono`, `rust_decimal`, `uuid`, `thiserror` dependencies to `crates/domain/Cargo.toml`. No framework or database dependencies.
2. Create the following modules in `crates/domain/src/`:
   - `models/` ‚Äî all domain structs
   - `error.rs` ‚Äî domain error types
   - `traits.rs` ‚Äî trait definitions
3. Define enums:
   - `AssetClass`: Stock, Bond, Commodity, Alternative, Crypto, CashEquivalent
   - `TransactionType`: Buy, Sell, Dividend, Coupon
4. Define domain structs (all with `id: Uuid`, `created_at: DateTime<Utc>`, `updated_at: DateTime<Utc>` where applicable):
   - `User` ‚Äî id, email, password_hash (implement Debug manually, mask password_hash)
   - `Portfolio` ‚Äî id, user_id, name, description, base_currency
   - `Asset` ‚Äî id, isin, yahoo_ticker (Option), name, asset_class, currency, exchange (Option)
   - `Transaction` ‚Äî id, portfolio_id, asset_id, transaction_type, date (NaiveDate), settlement_date (Option<NaiveDate>), quantity (Option<Decimal>), unit_price (Option<Decimal>), commission (Decimal), currency, exchange_rate_to_base (Decimal), gross_amount (Option<Decimal>), tax_withheld (Option<Decimal>), net_amount (Option<Decimal>), notes (Option), import_hash (Option)
   - `RefreshToken` ‚Äî id, user_id, token_hash, expires_at, revoked
   - `PriceRecord` ‚Äî id, asset_id, date (NaiveDate), close_price (Decimal), currency, source
   - `ExchangeRate` ‚Äî id, from_currency, to_currency, date (NaiveDate), rate (Decimal)
5. Define "new" structs for creation (without id/timestamps):
   - `NewUser`, `NewPortfolio`, `NewAsset`, `NewTransaction`, `NewRefreshToken`, `NewPriceRecord`, `NewExchangeRate`
6. Define domain error type with `thiserror`:
   - `DomainError` with variants: `NotFound(String)`, `ValidationError(String)`, `DuplicateEntry(String)`, `InsufficientQuantity { available: Decimal, requested: Decimal }`, `ExternalServiceError(String)`
7. Define pagination types:
   - `PaginationParams` ‚Äî page (u32), limit (u32)
   - `PaginatedResult<T>` ‚Äî data (Vec<T>), pagination (PaginationMeta)
   - `PaginationMeta` ‚Äî page, limit, total_items, total_pages

## Done criteria
- `cargo build` succeeds.
- `cargo doc --no-deps` builds without warnings.
- All public types have `///` doc comments.
- `User` struct has custom Debug impl that masks `password_hash`.
- No sqlx, axum, or infrastructure dependencies in `crates/domain/Cargo.toml`.
- All clippy/fmt checks pass.
